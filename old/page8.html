Инструкция

1. Зайдите в настройки и выберите проект;

2. Введите планируемое время по проекту в часах;

3. Нажмите кнопку "Сохранить";

4. Выберите сотрудника для фильтрации;

5. Выберите проект для фильтрации;

6. Посмотрите таблицу задач по проекту.

Так же присутствует функционал сортировки по времени и фильтрации по сотруднику.

Если фактическое затраченное время на проект больше чем то время, которые вы указали в настройках, то значение в строке "Планируемое время проекта" будет выделено красным.
